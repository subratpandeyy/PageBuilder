# Next.js Page Builder (Puck)

A simple page builder built with **Next.js**, **Tailwind CSS v4**, and **Puck**.  
This project demonstrates how to create a **design-in-browser editor** that can be integrated with any CMS.

---

## Stack
- Next.js
- Tailwind CSS v4
- Puck
- TypeScript

---

## Run Locally
```bash
npm install
npm run dev
